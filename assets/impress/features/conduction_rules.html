The input material

To demonstrate, what the conduction rules are capable of, we start, with this High Hat pattern.
Without any articulation, the pattern sounds very mechanical.
So lets add some articulation to that.

Adding articulation

One way to do so, is adding velocity commands before every high hat note.
\pp h16 \pp h \ff h \pp h \pp h \pp h \ff h \pp h \pp h \pp h \ff h \pp h \pp h \pp h \ff h \pp h
Technically, this would be ok. But the sheet code, is now very hard to read.
In order to keep the sheet source clean, we want to separate the velocity information from the actual sheet. If you familiar with html and css, you already know the concept. We use a separate file, to describe, how to perform the notes of the sheet.

The first step is to select the notes we want to affect. In this case, we want to emphasize the high hat notes on the off beat. Therefore, we select, the instrument, drums.
then, we select, the pitch, h. Which is related to the high hat.
we select, the notes, on beat .5. Which addresses the notes on the off beat.
And finally, we describe, what we want to do. In this case we want to emphasize. So we add 50 velocity units, to the selected notes.
